# Aftersales Service Analytics Dashboard

This project analyzes vehicle service data to identify common vehicle problems, repair solutions, and service trends across cities and vehicle companies.

## Tools Used
Python
Pandas
Matplotlib
Power BI

## Project Workflow
1. Data collection and preprocessing using Python
2. Exploratory data analysis
3. Visualization using Power BI
4. Interactive dashboard for service analytics

## Key Insights
- Engine and brake issues appear most frequently in service records.
- Battery replacement and brake repair are common solutions.
- Service requests are concentrated in major cities.
- Some vehicle companies show higher service demand.
